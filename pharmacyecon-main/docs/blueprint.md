# **App Name**: MediLink Lite

## Core Features:

- Medicine Finder: Allow users to search for medicines from different pharmacies.
- Medicine Details: Display detailed information about each medicine, including price, stock, pharmacy details, and expiration date.
- Pharmacy Listing: Display a list of registered pharmacies.
- Prescription Upload: Securely upload prescriptions for prescription-based orders.
- AI-Driven Recommendations: Provide AI-driven medicine recommendations based on user search history. The tool suggests relevant alternatives to prescribed medication using an LLM.
- User Dashboard: Provide a dashboard where users can view order history, prescriptions, and account information.
- Pharmacy Admin Panel: Pharmacies can manage their medicines and orders.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and health.
- Background color: Light blue (#E3F2FD) for a clean, professional feel.
- Accent color: Soft green (#A5D6A7) for call-to-actions related to health.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible readability.
- Use modern, minimalist icons for medicine categories and actions.
- Responsive grid layout using Bootstrap to ensure compatibility across devices.
- Subtle transitions and animations to improve user experience (e.g., loading indicators, confirmation messages).
